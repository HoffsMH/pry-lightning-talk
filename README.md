### Things to install
* ## Pry-toys
  * for when you need to test something real quick
    ```bash
     gem install pry-toys
     ```
     #### Try
     ```ruby
    Hash.toy(300)
    Array.toy(3, Time)
     ```

* ## Pry-byebug
 * for stepping through code
   ```bash
    gem install pry-byebug
    ```
    #### Try
    ```ruby
   step # Moves through code line by line
   finish #completes rest of cod
    ```
